# Abstracción

La abstracción es una propiedad de la programación orientada a objetos que permite identificar solo las características esenciales de un objeto, ignorando lo que no es necesario.
Esto ayuda a simplificar el código y enfocarse en lo importante.

## Ejemplo en Python:

from abc import ABC, abstractmethod

class Animal(ABC):
    @abstractmethod
    def hacer_sonido(self):
        pass

class Perro(Animal):
    def hacer_sonido(self):
        return "Guau"

mi_perro = Perro()
print(mi_perro.hacer_sonido())
